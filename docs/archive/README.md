This folder holds historical planning and rollout artifacts that no longer belong in the main `docs/` surface.

Use the top-level `docs/` folder for current truth:

- `docs/ASSISTANT-LED-ROADMAP.md`
- `docs/ASSISTANT-LED-HISTORY-SUMMARY.md`
- `docs/PROGRAM-COMPLETE-SUMMARY.md`
- `docs/2026-03-24-system-audit.md`
- `docs/AUTOMATIONS.md`
- `docs/NEW-MACHINE-SETUP.md`

Use this archive when you need the detailed delivery trail behind that current state:

- `assistant-led-phases/`
  Assistant-led phase plans, rollouts, audits, and terminal Phase 37/38 artifacts.
- `legacy-program/`
  The earlier `PHASE-*` planning and rollout sequence that led into the later assistant-led track.
- `post-launch/`
  The post-launch phase plans, handoffs, and rollout records that sit between the legacy program and the assistant-led track.

The archive is intentionally preserved as historical reference. It is not the recommended onboarding path for new operators or contributors.
